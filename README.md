# JERRY-Voice-Assistant-System

Jerry is a Python-based voice assistant that can interact with the user through speech recognition, perform tasks like opening applications, searching on Wikipedia, playing music randomly, and having small talk.

This project uses speech recognition and text-to-speech (TTS) to provide a hands-free assistant experience similar to Iron Man's Jerry.

- Recognize voice commands using Google Speech Recognition

- Speak responses using pyttsx3

- Time & Date announcements

- Wikipedia search with spoken summary

- Open websites like Google, Linkedin, YouTube

- Play random music from a specified folder

- Respond to basic small talk

- Shutdown the VS code.

- Shutdown the computer.

- Exit gracefully with a voice command

## How to run? 

1. Create a virtual environment:

   ```bash
   conda create -n Jerry python=3.11 -y
   ```

2. Activate virtual environment:

   ```bash
   conda activate Jerry
   ```

3. Install required packages:

   ```bash
   pip install -r requirements.txt
   ```

4. Run the Jerry script:

   ```bash
   python jerry.py
   ```
